# Coding Challenge - Tic-Tac-Toe

This branch contains the code and documentation for a simple implementation of the Tic-Tac-Toe game in Python. 


| File / Folder|Description|
| -------------| ----------|
| [**01 - Pseudocode**](03_intro_to_pseudocode.md)| Pseudocode for the Tic-Tac-Toe game.|
| [**02 - Flowchart**](02_flowchart.md)| Flowchart that breaks down the logic of the Tic-Tac-Toe game. |
| [**Tic Tac Toe**](tic_tac_toe.py)| Implementation of the Tic-Tac-Toe game in Python.|

## Usage
You can run the game using the following command in your terminal:

   ```bash
   uv run python tic_tac_toe.py
   ```
