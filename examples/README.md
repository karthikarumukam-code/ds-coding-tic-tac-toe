# Examples

These two scripts, `script1.py` and `script2.py`, demonstrate how:

```python
if __name__ == "__main__":
```

works in Python.

For each script:

1. Read through the code.
2. Predict what the output will be.
3. Run the script:

   ```bash
   uv run python script1.py
   ```

   Then repeat with:

   ```bash
   uv run python script2.py
   ```
4. Compare the actual output with your prediction.

Pay attention to what changes when a file is executed directly versus when it is imported.
